## Используемые пины и что к ним подключать

```
  ╔══════════════════════════════════════════════════════════╗
  ║              FPV MultiTool — Разводка                   ║
  ╠══════════════════════════════════════════════════════════╣
  ║                                                          ║
  ║  GPIO 2 ──── SIGNAL (серво / ESC мотор)                 ║
  ║  GPIO 43 ─── UART TX → к RX приёмника ELRS             ║
  ║  GPIO 44 ─── UART RX → к TX приёмника ELRS             ║
  ║  GPIO 3 ──── BOOT ELRS → к GPIO0 приёмника             ║
  ║  5V ──────── Питание ELRS приёмника                     ║
  ║  GND ─────── Общая земля (ОБЯЗАТЕЛЬНО!)                 ║
  ║                                                          ║
  ║  GPIO 48 ─── I2C SDA → к SDA батареи DJI               ║
  ║  GPIO 47 ─── I2C SCL → к SCL батареи DJI               ║
  ║  GND ─────── к GND батареи DJI                          ║
  ║                                                          ║
  ╚══════════════════════════════════════════════════════════╝
```

## 1. Подключение Servo / ESC мотора

```
     ESP32-S3                    Servo / ESC
    ┌────────┐                  ┌──────────┐
    │        │                  │          │
    │  GPIO 2├──── Signal ─────►│ SIG      │ (белый/жёлтый)
    │        │                  │          │
    │    GND ├──── GND ────────►│ GND      │ (чёрный/коричневый)
    │        │                  │          │
    └────────┘         ┌───────►│ VCC      │ (красный)
                       │        └──────────┘
                  ┌────┴────┐
                  │ Батарея │  5V (серво)
                  │  или    │  или
                  │  BEC    │  2-6S (ESC)
                  └─────────┘
    
    ⚠ Серво/ESC питается ОТДЕЛЬНО, не от ESP32!
    ⚠ Общий GND обязателен!
```

## 2. Подключение ELRS приёмника (прошивка)

```
     ESP32-S3                   ELRS Receiver
    ┌────────┐                 ┌─────────────┐
    │        │                 │             │
    │GPIO 43 ├──── TX ────────►│ RX          │
    │  (TX)  │                 │             │
    │GPIO 44 ├──── RX ◄────────│ TX          │
    │  (RX)  │                 │             │
    │ GPIO 3 ├──── BOOT ──────►│ GPIO0/BOOT  │
    │        │                 │             │
    │    5V  ├──── 5V ────────►│ VCC (5V)    │
    │        │                 │             │
    │   GND  ├──── GND ───────►│ GND         │
    │        │                 │             │
    └────────┘                 └─────────────┘
    
    ⚠ TX и RX перекрёстно! ESP TX → Receiver RX
    ⚠ ELRS модули толерантны к 5V — питаем с платы
    ⚠ BOOT нужен только для входа в режим прошивки
```

## 3. Подключение батареи DJI Mavic 3

```
     ESP32-S3                   DJI Battery (8 контактов)
    ┌────────┐                 ┌─────────────────────────┐
    │        │                 │ 1  2  3  4  5  6  7  8  │
    │GPIO 48 ├── SDA ─────────►│          ▲               │
    │  (SDA) │      ┌─[4.7K]──┤          │               │
    │        │      ├── 3.3V   │       SDA SCL            │
    │GPIO 47 ├── SCL ─────────►│              ▲           │
    │  (SCL) │      ├─[4.7K]──┤              │           │
    │        │      └── 3.3V   │                          │
    │   GND  ├── GND ─────────►│ ▲  ▲  ▲                 │
    │        │                 │ │  │  │     ▲  ▲  ▲     │
    └────────┘                 │GND GND GND  V+ V+ V+    │
                               └─────────────────────────┘

    Вид на контакты батареи (снизу):
    ┌─────────────────────────────────────┐
    │  1    2    3    4    5    6   7   8  │
    │ GND  GND  GND  SDA  SCL  V+  V+  V+│
    └─────────────────────────────────────┘

    ⚠ НЕ ПОДКЛЮЧАТЬ V+ (15.4V) к ESP32 — сгорит!
    ⚠ Только SDA + SCL + GND
    ⚠ Pull-up 4.7K к 3.3V на обеих линиях
    ⚠ I2C адрес батареи: 0x0B
```

## 4. Сводная таблица пайки

```
    ┌──────────┬──────────┬───────────────────────────┐
    │ GPIO     │ Провод   │ Куда                      │
    ├──────────┼──────────┼───────────────────────────┤
    │ GPIO 2   │ Жёлтый   │ Signal серво/ESC          │
    │ GPIO 43  │ Зелёный  │ TX → RX приёмника ELRS    │
    │ GPIO 44  │ Белый    │ RX ← TX приёмника ELRS    │
    │ GPIO 3   │ Оранж.   │ BOOT → GPIO0 ELRS        │
    │ GPIO 48  │ Синий    │ SDA → Pin4 DJI батареи    │
    │ GPIO 47  │ Фиолет. │ SCL → Pin5 DJI батареи    │
    │ 5V       │ Красный  │ VCC ELRS приёмника        │
    │ GND      │ Чёрный   │ GND (общий для всего)     │
    └──────────┴──────────┴───────────────────────────┘
```


## Что НЕ нужно подключать

- **GPIO 0** — кнопка BOOT на плате (навигация по меню)
- **GPIO 38** — RGB LED (на плате)
- **GPIO 1** — ADC батареи платы (на плате)
- **GPIO 14-18, 21** — SD карта (на плате)
- **GPIO 39-42, 45-46** — LCD дисплей (на плате)
- **GPIO 47-48** — I2C (на плате, НО используется и для батареи DJI)
- **GPIO 19-20** — USB (НЕ ТРОГАТЬ!)
