Switch boot slot non-destructive

Изменить otadata так чтобы при следующем boot грузился другой slot. Полезно если активный slot убит (бесконечный crash loop). Текущий: app1 (MILELRS). Переключить на app0 (ELRS 3.5.3)?

Erase OTADATA destructive

Erase 8 KB at 0xE000. ESP-IDF bootloader fallback на app0 при пустой otadata. Не трогает сами app-партиции. Защита: type "ERASE OTADATA" в modal.

Erase partition destructive

Wipe entire partition (chunked 64 KB blocks, single ROM session). Choose what:

Full flash dump read-only

Read 4 MB SPI flash → save as dump_YYYY-MM-DD.bin. ~9 min @ 115200 (ROM CMD_READ_FLASH_SLOW). Stub mode: ~1 min (CMD_READ_FLASH 0xD2 streaming).

Restore from backup destructive

Upload a previous dump.bin → flash back at offset 0x000000 (full image including bootloader). Use case: undo botched experiment. Backup must match this RX's chip + flash size — dump's MAC is checked.
