Mockup mode. Это статичный preview нового UI без подключения к плате. Открой Battery, Receiver, Motor, OTA, USB, RC Sniff, Servo, Setup, System — итерируем визуал и IA, потом переносим в живой код.
Принципы
  • Goal-driven IA — наверху каждого таба «Что ты хочешь сделать?» с decision-cards (как в Setup)
  • One thing at a time — выбрал действие → увидел его экран. Остальные feature-knobs скрыты в Advanced.
  • Status panel снизу — что сейчас активно / что в NVS на следующий boot
  • Apply gate — destructive действия требуют подтверждения через modal
  • No inline styles — всё через design tokens + utility-классы
