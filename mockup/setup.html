Один вопрос за раз: что подключаешь к Port B, кто работает. Остальное (USB descriptor, electrical mode) автоматически.
1 · What's connected to Port B?
🔋 Battery DJI / SBS battery pack on SMBus.
📡 Receiver ELRS / CRSF receiver on UART.
⚡ Motor / Servo ESC DShot / Servo PWM on signal pin.
🔌 Advanced PPM / raw GPIO sniffer / custom protocol.
💤 Idle Nothing connected, pins floating.
2 · Who interacts with it?
🖥 HOST — plate itself Web UI читает данные, плата сама общается с устройством. CDC USB.
💻 BRIDGE — PC via plate Прокидываю на компьютер: bqStudio / esptool / Battery Killer / любая SMBus tool.
→ Port B = UART, USB = CDC

Будет применено: Port B → UART (TX=GP11, RX=GP10), USB → CDC (Serial). Reboot не нужен (Port B меняется на лету). Сохранится в NVS.
